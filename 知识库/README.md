# 知识库

存放交易知识沉淀与策略源材料。

## 策略源（大神摘录）

`策略源/` 下可按「大神名」或「策略名」建子目录，放入其文章、语录的原始摘录（Markdown 或文本）。  
需要生成或更新 Cursor 策略包时，在对话中请求 AI 根据该目录内容做**一次性策略总结**，并写入 `.cursor/skills/strategy-<名称>/SKILL.md` 的「本策略规则」部分。不做持续增量更新；若后续有新增摘录，可再次执行总结并手动更新对应 Skill。
